HTML CSS JS PHP SQL

HTML

Linguagem de marcação para estruturar o conteúdo de páginas web, definindo elementos e sua organização.

CSS

Linguagem de estilo para controlar a apresentação visual das páginas HTML, como cores, fontes e layout.

Javascript

Linguagem de programação para adicionar interatividade e dinamismo às páginas web, executada no navegador.

PHP

Linguagem de script do lado do servidor, usada principalmente para desenvolver aplicações web dinâmicas e interagir com bancos de dados.

SQL

Linguagem de consulta estruturada utilizada para gerenciar e manipular dados em bancos de dados relacionais.
